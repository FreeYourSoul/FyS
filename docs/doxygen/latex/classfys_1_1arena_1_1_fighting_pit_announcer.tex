\doxysubsection{Detailed Description}
Builder class to create the \mbox{\hyperlink{classfys_1_1arena_1_1_fighting_pit}{Fighting\+Pit}}. 


\begin{DoxyItemize}
\item It generate the position of the Pit participants (contenders and team)
\item Define if the fight is an ambush or not.
\item Download the required scripts for the contenders (if not in the cache).
\item Download the required scripts for the team (if not in the cache)
\item Set the level of the fighting pit 
\end{DoxyItemize}
