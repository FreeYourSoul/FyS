\doxysubsection{Detailed Description}
Layout of the fighting pit. Manage the positions of monsters and the players/monsters movements. 

This class is mapping the characters (contenders or N\+PC) with the layout of the fightingpit. The layout of the fighting pit can be taken as a draughtboard composed of 3 hexagons as shown as below
