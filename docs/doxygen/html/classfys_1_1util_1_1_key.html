Detailed Description

Key used as token At creation will create a token of 8 characters that will be initialized using the address of the Key object as base. Hide the array index (index that the player data are in) inside the token.

The encoding follow this rule: <offset until="" index=""> <random> <index> one char KEY_SIZE - (1 + index size) (KEYSIZE - offset)
