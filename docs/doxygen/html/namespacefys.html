Get the most suitable (suitable being defined by a comparator) of a given container. The most suitable follow the comparator.

For example the most suitable following this comparator [](auto currentSuitable, auto next) { return currentSuitable < next; } will be the biggest element of the container.

Parameters
firstiterator start of the container
lastiterator end of the container
predicatepredicate that takes the current most suitable and the next value to do a comparison
Returns
iterator pointing on the maximum value described by the predicate, if container is empty, last is returned
