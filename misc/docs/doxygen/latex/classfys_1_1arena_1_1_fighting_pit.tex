\doxysubsection{Detailed Description}
A fighting pit represent a specific instance of a battle. 

A fightingpit is composed of\+:
\begin{DoxyItemize}
\item a layout (mapping of how the monsters/players are spread on the board)
\item a \mbox{\hyperlink{classfys_1_1arena_1_1_pit_contenders}{Pit\+Contenders}} object managing all the monsters and their behaviours (scripting)
\item a \mbox{\hyperlink{classfys_1_1arena_1_1_ally_party_teams}{Ally\+Party\+Teams}} object managing the players, each client has a \mbox{\hyperlink{classfys_1_1arena_1_1_party_team}{Party\+Team}} (multiple party teams possible for one \mbox{\hyperlink{classfys_1_1arena_1_1_ally_party_teams}{Ally\+Party\+Teams}}) and the \mbox{\hyperlink{classfys_1_1arena_1_1_party_team}{Party\+Team}} is composed of \mbox{\hyperlink{classfys_1_1arena_1_1_team_member}{Team\+Member}} that are the actual characters spread on the gaming board.
\item The 18 instances of \mbox{\hyperlink{classfys_1_1arena_1_1_side_battle}{Side\+Battle}}
\end{DoxyItemize}
